Cara Hitung Subnet IP CIDR Kelas C

Contoh soal:

203.45.118.173/28

Cari:

  1. Subnet mask
  2. Jumlah host yang bisa dipakai
  3. IP Network
  4. IP Broadcast

1. Cari subnet mask

  • /8 = 255.0.0.0
  • /16 = 255.255.0.0
  • /24 = 255.255.255.0
  • +1 = 128
  • +2 = 192
  • +3 = 224
  • +4 = 240
  • +5 = 248
  • +6 = 252
  • +7 = 254

Jadi:

/28 = /24 + 4

/28 = 255.255.255.0 + 240

/28 = 255.255.255.240

2. Konversikan subnet mask ke biner

Cara mengkonversikannya adalah mengurangi setiap oktet dengan 2 yang dipangkatkan dengan suatu bilangan, dimulai dari 27 atau 128 sampai 20 yaitu 1.

Bilanganya adalah sebagai berikut:

128 64 32 16 8 4 2 1

Dimulai dari oktet pertama:

255 - 128 = 127

127 - 64 = 63

63 - 32 = 31

31 - 16 = 15

15 - 8 = 7

7 - 4 = 3

3 - 2 = 1

1 - 1 = 0

Jika bisa dikurangi dengan bilangan tersebut, maka binernya adalah 1. Jika tidak bisa, binernya 0.

1 1 1 1 1 1 1 1
128 64 32 16 8 4 2 1

Jadi 255 binernya adalah 1111111. Karena oktet ke-2 dan 3 sama, langsung saja skip ke oktet 4.

Oktet 4:

240 - 128 = 112

112 - 64 = 48

48 - 32 = 16

16 - 16 = 0

1 1 1 1 0 0 0 0
128 64 32 16 8 4 2 1

Jadi 240 binernya adalah 11110000.

Jadi subnet mask 255.255.255.240 binernya adalah 11111111.11111111.11111111.11110000.

Cara cepat:

Perhatikan tabel berikut:

Oktet Biner
0 00000000
128 10000000
192 11000000
224 11100000
240 11110000
248 11111000
252 11111100
254 11111110
255 11111111

Dari tabel tersebut, dapat dengan mudah diketahui bahwa biner dari subnet mask 255.255.255.240 adalah 11111111.11111111.11111111.11110000.
