1. Cari subnet

Udah tau lah ya. Kalo belum tau:

  • /8 = 255.0.0.0
  • /16 = 255.255.0.0
  • /24 = 255.255.255.0
  • +1 = 128
  • +2 = 192
  • +3 = 224
  • +4 = 240
  • +5 = 248
  • +6 = 252
  • +7 = 254

Jadi:

/28 = /24 + 4

/28 = 255.255.255.0 + 240

/28 = 255.255.255.240
